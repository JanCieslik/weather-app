A responsive React weather application that allows users to track weather conditions for multiple cities, save favorite locations, and switch between temperature units. Built with modern frontend tools and best practices, the project demonstrates practical experience with React, state management, routing, and API integration.

## Features

- Display weather data for multiple cities using OpenWeather API
- Add and remove weather widgets dynamically
- Save favorite cities to local storage
- Temperature unit toggle: Celsius, Fahrenheit, Kelvin
- Responsive design using Bootstrap and Tailwind
- Client-side routing with React Router
- Random city suggestions on initial load

## Tech Stack

- React 18
- React Router DOM
- JavaScript (ES6+)
- HTML5 & CSS3
- Bootstrap 5 & TailwindCSS
- OpenWeather API
- Vite.js for fast development and build
- GitHub Pages for deployment
